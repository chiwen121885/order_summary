# 訂單整理(order_summary)
將官網訂單整理成內部雲端格式
python+excel

# 起因
1. 官網訂單需要一筆筆點進去，才可以看到訂單內容
2. 每筆訂單需要手動key入google sheet，包含日期、品項、付款方式、送貨方式、備註
3. 每筆訂單需要手動計算key入金額
4. 手動key訂單，花時間又易出錯，導致後面客訴處理

# 功能
1. 將官網匯出的excel訂單格式，1秒轉換成雲端格式
2. 不需手key訂單內容及金額，節省時間
3. 所有訂單一目瞭然，可快速判斷可否出貨
